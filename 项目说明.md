# 基于轻量化AI算法的红外多目标检测与跟踪方法研究

## 📋 项目概述

本项目是西北农林科技大学信息工程学院2026届本科毕业设计，旨在研究并实现一套基于轻量化AI算法的红外多目标检测与跟踪系统，最终部署在Rockchip RV1126嵌入式平台上。

### 项目目标
- **检测性能**：mAP@0.5 ≥ 75%，Precision ≥ 80%，Recall ≥ 85%
- **跟踪性能**：MOTA ≥ 60%，IDF1 ≥ 65%
- **系统性能**：检测/跟踪 ≥ 25FPS，超分/去噪 ≥ 50FPS，端到端延迟 ≤ 50ms

### 主要研究内容
1. YOLO算法在热红外图像处理中的应用及其轻量化研究
2. 基于YOLO算法的热红外图像多目标检测与跟踪方法
3. 基于Rockchip RV1126的红外多目标检测与跟踪系统实现
4. 系统运行与测试

## 🚀 快速开始

### 环境要求
- Python 3.8+
- PyTorch 1.10+
- CUDA 11.0+ (GPU训练)
- RV1126开发板 (嵌入式部署)

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/Coolzs77/bishe.git
cd bishe

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux
# 或 venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 下载数据集
python scripts/data/download_dataset.py

# 5. 准备数据
python scripts/data/prepare_flir.py --input data/raw/flir
```

### 训练模型

```bash
# 训练YOLOv5s基准模型
python scripts/train/train_yolov5.py --config configs/train_config.yaml

# 运行消融实验
python scripts/train/train_ablation.py
```

### 评估模型

```bash
# 评估检测性能
python scripts/evaluate/eval_detection.py --weights outputs/weights/best.pt

# 评估跟踪性能
python scripts/evaluate/eval_tracking.py --tracker deepsort

# 对比不同跟踪算法
python scripts/evaluate/compare_trackers.py
```

### 部署到RV1126

```bash
# 导出ONNX模型
python scripts/deploy/export_model.py --weights outputs/weights/best.pt

# 转换为RKNN格式
python scripts/deploy/convert_to_rknn.py --onnx outputs/weights/best.onnx

# 测试RKNN模型
python scripts/deploy/test_rknn.py
```
