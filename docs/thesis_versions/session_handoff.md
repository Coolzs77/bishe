## 关键上下文（供后续会话使用）

### 核心实验结论

1. **全消融最优**：exp07 EIoU，mAP@0.5=0.817（+0.85pp vs. Baseline），零参数增量
2. **最优跟踪器**：ByteTrack，ID Switch=26（exp07检测源），DeepSORT=128，CenterTrack=87
3. **板端FPS**：含跟踪27.6 FPS，纯检测≥33 FPS（5项优化累计+31%）
4. **板端跟踪优化**：R2配置，唯一ID↓14%，展示覆盖率↑18%
5. **轻量化备选**：exp09 Ghost+EIoU，参数量↓30.3%，mAP仅↓1.9pp
