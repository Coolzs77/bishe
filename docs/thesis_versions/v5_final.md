## 四、Word排版执行指南

### 4.1 Markdown到Word的映射规则

| Markdown元素 | Word格式 |
|-------------|---------|
| `# 标题` | 封面标题，黑体二号加粗居中 |
| `## 第X章` | 章标题，黑体三号加粗居中，前后各空一行 |
| `### X.X` | 节标题，黑体四号加粗左对齐 |
| 正文段落 | 宋体/TNR小四号，首行缩进2字符 |
| `$...$` | 公式，居中显示，右对齐编号(X-X) |
| `**【图X-X】**` | 图标题，宋体五号居中，置于图下方 |
| `**【表X-X】**` | 表标题，宋体五号居中，置于表上方 |
| `[N]` | 上角标引用，如"YOLOv5^[15]" |
| `**【证据】**` | 删除（仅供草稿溯源，不出现在终稿中） |
| 代码块 | 附录中保留，正文中删除或改为伪代码 |

### 4.2 排版顺序

1. 封面（按学院模板）
2. 中文摘要+关键词（独立页）
3. 英文Abstract+Keywords（独立页）
4. 目录（自动生成）
5. 正文第1-8章
6. 参考文献
7. 致谢（盲审版省略或匿名化）
8. 附录A
9. 附录B
