# 基于轻量化方法的红外多目标检测与跟踪研究

## 项目概述

本项目围绕热红外场景下的多目标检测与跟踪展开，当前仓库聚焦以下研究任务：

- 改进 YOLOv5 检测模型并开展消融实验
- 在统一检测器基础上对比 DeepSORT、ByteTrack、CenterTrack
- 汇总检测与跟踪结果，为论文实验部分提供依据

部署、RKNN 转换和嵌入式代码已从当前仓库主线移除。

## 当前保留的模块

- configs: 数据、训练、跟踪配置
- data: 处理后的数据与热视频
- scripts/train: 训练与消融脚本
- scripts/evaluate: 检测评估、跟踪评估、结果绘图
- src/detection: 检测器实现
- src/tracking: 跟踪器实现
- src/evaluation: 指标与汇总逻辑
- outputs: 实验输出

## 当前实验流程

### 1. 数据准备

整理 FLIR 数据集与热视频数据，形成统一输入目录。

### 2. 检测训练

使用 train_yolov5.py 或 train_ablation.py 完成基准模型与消融实验训练。

### 3. 检测评估

使用 eval_detection.py 统一比较各消融模型的 Precision、Recall、mAP@0.5、mAP@0.5:0.95。

### 4. 跟踪评估

选择候选检测模型后，用 eval_tracking.py 分别评估 deepsort、bytetrack、centertrack。

### 5. 结果汇总

使用 plot_eval_summary.py 和 plot_tracking_algorithm_comparison.py 输出图表与汇总表。

## 当前结论边界

- 当前数据配置以 train 和 val 为主，没有独立 test 集。
- 因此检测指标默认属于验证集结论。
- 跟踪结果依赖于所选检测模型，应先固定检测模型再比较 tracker。

## 当前推荐使用的脚本

### 检测批量评估

```bash
python scripts/evaluate/eval_detection.py --mode metric --batch-eval --ablation-dir outputs/ablation_study --stage all --weights-name best.pt --data data/processed/flir/dataset.yaml --img-size 640 --batch-size 16 --device 0 --workers 8 --save-csv --sort-by map5095
```

### 跟踪评估

```bash
python scripts/evaluate/eval_tracking.py --weights outputs/ablation_study/ablation_exp07_eiou/weights/best.pt --data data/videos/thermal_test --tracker deepsort --output outputs/tracking/current --device 0 --img-size 640 --half --no-save-vid --no-save-txt --no-overlay
```

### 跟踪绘图

```bash
python scripts/evaluate/plot_tracking_algorithm_comparison.py --base-dir outputs/tracking/rerun_20260324_exp07
```
